## Overview

This repository hosts the source code for my Virtual Portfolio — a personal, interactive website that consolidates my qualifications, projects, and skills into one accessible place. It's designed to serve as a living CV I can point employers, recruiters, and collaborators to, without needing to send a static PDF.

The site is built entirely from scratch using core web technologies, with a focus on clean navigation and quick access to the information a hiring manager actually wants to see.

---

## Site Sections

| Section | What it Covers |
|---|---|
| About Me | A personal introduction — who I am, my interests, and what drives me in tech |
| CV | Formal qualifications, with a focus on my education and academic progress |
| Projects | A curated showcase of the projects I've completed throughout university, each linked to its own GitHub repository |
| Skills | Programming languages, IT concepts, and soft skills, organized for quick scanning |
| Contact | Direct links to reach me — email, phone, and LinkedIn |

---

## Key Features

- Download CV button — instantly download the full CV as a PDF
- Tab-strip navigation — move between sections without page reloads
- Quick contact access — a dedicated button jumps straight to contact details
- Linked project repositories — every project links directly to its GitHub source
- Clickable contact details — email, phone, and LinkedIn are all directly actionable

---

## Getting Started

Clone the repository to run it locally:

```bash
git clone https://github.com/NathanFE05/Virtual-Portfolio.git
cd Virtual-Portfolio
```

Then open `index.html` in your browser — no build tools or dependencies required.
